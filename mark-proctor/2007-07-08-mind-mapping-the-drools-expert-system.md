When talking people through the Drools 4.0 set of features and how they work, I’ve found that there is simply too much for people to grok, without previous Expert System experience. So I thought I’d turn to [Mind Maps](<http://en.wikipedia.org/wiki/Mind_map>) to see if I can help make all our advanced concepts more grokkable. I found a great GPL tool called [FreeMind.](<http://freemind.sourceforge.net/wiki/index.php/Main_Page>) I’m using version 9-beta9, which seems to be stable enough to use. I’ve put down an initial brain dump to get started, the idea is map all our concepts with notes and links through to the main documentation or other related sites for more info.  
[![](/legacy/assets/images/2007/07/bc369cba3291-drools-mindmap.PNG)](</assets/images/2007/07/e720237165ff-drools-mindmap.PNG>)The FreeMind file for the above map is in subversion, the documentation/mindmap folder, FreeMind also does flash exports, which I’ve committed, so you can play with it live now by clicking [here](<http://anonsvn.labs.jboss.com/labs/jbossrules/trunk/documentation/mindmap/drools.html>), and give me your feedback.
