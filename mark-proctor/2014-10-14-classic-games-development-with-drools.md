### [Classic Games Development with Drools](<https://blog.kie.org/2014/10/classic-games-development-with-drools.html>)

by [Mark Proctor](<https://blog.kie.org/category/all?search_authors=3> "Posts by Mark Proctor") \- October 14, 2014  
[Rules](<https://blog.kie.org/category/rules>) [Article](<https://blog.kie.org/content_type/article>)

I realised I didn’t upload my slides from [Decision Camp](<http://www.decision-camp.com/>) 2013, where I talked about using Games to learn Rule Based Programming. Sorry about that, here they are better late than never:  
[Learning Rule Based Programming Using Games](<http://www.slideshare.net/MarkProctor/games-40228109>)

The talk provides a gentle introduction into rule engines and then covers a number of different games, all of which are available to run from the drools examples project.

  * Number Guess
  * Adventure Game
  * Space Invaders
  * Pong
  * Wumpus World

While there is no video for the presentation I gave, I have made videos for some of these games in the past. Although beware some of them may be a little out of date now, compared to the version that is in our current examples project.
