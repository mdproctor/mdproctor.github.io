Drools 4.0 downloads are going strong for August, its only the 22nd, and for this month we now have (K is for downloads in the thousands, 12.5K is 12500 downloads):
[code]
    12.5K drools-4.0.0-bin.zip  
    6.5K  drools-4.0.0-brms-standalone.zip  
    5.3K  drools-4.0.0-src.zip  
    5K    drools-4.0.0-eclipse.zip  
    3.7K  drools-4.0.0-brms.zip  
    3.4K  drools-4.0.0-examples.zip  
    
[/code]

Just a reminder that drools can be downloaded here <http://labs.jboss.com/drools/downloads.html> which also provides details of the Eclipse update site which is here <http://downloads.jboss.com/drools/updatesite>. The manual also gives details on how to use the Eclipse update site which can be found [here](<http://downloads.jboss.com/drools/docs/4.0.0.13773GA/html_single/index.html#d0e1325>).

Drools 4.0.1 is out the end of this week. We’ve worked furiously to fix a lot of errors and things are now looking very stable, thanks to the great community feedback we have received. Once that is out we’ll branch and the exciting work for the next major release will begin.
