### [JNI4Net now has Drools example](<https://blog.kie.org/2011/02/jni4net-now-has-drools-example.html>)

by [Mark Proctor](<https://blog.kie.org/category/all?search_authors=3> "Posts by Mark Proctor") \- February 28, 2011  
[Rules](<https://blog.kie.org/category/rules>) [Article](<https://blog.kie.org/content_type/article>)

We often get asked about running Drools and jBPM on .net.

Today [FrankVhh](<http://drools-java-rules-engine.46999.n3.nabble.com/template/NamlServlet.jtp?macro=user_nodes&user=304907>) pointed out, in this [post](<http://drools-java-rules-engine.46999.n3.nabble.com/Drools-NET-active-tp2575350p2594095.html>), that there has been some success using just pure IKVM. Frank pointed to the previous post by [Corneil du Plessis](<http://drools-java-rules-engine.46999.n3.nabble.com/template/NamlServlet.jtp?macro=user_nodes&user=4855>), which can be found here:  
<http://drools-java-rules-engine.46999.n3.nabble.com/Drools-Java-and-Net-td2290536.html#a2414244>

He also pointed out success with the [jni4net](<http://jni4net.sourceforge.net/>) project, which now comes with a Drools sample, to help you get started.  
<http://jni4net.sourceforge.net/>
