### [Properties Widget In Guvnor](<https://blog.kie.org/2008/08/properties-widget-in-guvnor.html>)

by [Mark Proctor](<https://blog.kie.org/category/all?search_authors=3> "Posts by Mark Proctor") \- August 12, 2008  
[Tools](<https://blog.kie.org/category/tools>) [Article](<https://blog.kie.org/content_type/article>)

To continue the [last post](<http://blog.athico.com/2008/08/more-widgets-for-guvnor-anton-arhipov.html>) on this topic, the properties editor widget is complete and is in Guvnor code base from now on.

[![](/legacy/assets/images/2008/08/73cb2a611022-properties.png)](<http://2.bp.blogspot.com/_flYJTi1O_TE/SKIh0aUEoqI/AAAAAAAAFvA/pe2rdKC4NJs/s1600-h/properties.png>)  
With this widget one can upload a classic properties file, edit it, and also download it back.
