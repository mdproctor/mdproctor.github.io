### [Drools Flow view in Guvnor with GWT Diagram](<https://blog.kie.org/2008/07/drools-flow-view-in-guvnor-with-gwt-diagram.html>)

by [Mark Proctor](<https://blog.kie.org/category/all?search_authors=3> "Posts by Mark Proctor") \- July 23, 2008  
[Process](<https://blog.kie.org/category/process>) [Article](<https://blog.kie.org/content_type/article>)

We now have Guvnor displaying uploaded Drools Flow files. We use the native GWT XML api to read in the XML document and GWT Diagram to generate the diagram. At the moment it’s view only, over time we will make it editable – which will sure beat round tripping with Visio that other platforms do.
