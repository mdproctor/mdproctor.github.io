The new [UberFire](<http://www.uberfireframework.org/>) form builder, that will be part of the jBPM 7.0 distribution, is making great progress. Underneath it is a [Bootstrap](<http://getbootstrap.com/css/>) grid system, but it addresses the issue of other Bootstrap layout builders that require the user to explicit add the grid layout first. Instead it dynamically alters the underlying grid as the user drags and places the components. The same builder code will be used for the latest DashBuilder dashboards too. There are more CSS improvements to come, but you can watch a video below (don’t forget to turn on HD and watch it full screen), demonstrating nested form capabilities. Eventually you should be able to build and deploy these types of applications live on [OpenShift](<https://www.openshift.com/>). Good work Pere and Eder.
