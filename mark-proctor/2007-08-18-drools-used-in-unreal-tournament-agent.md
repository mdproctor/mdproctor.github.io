[Pogamut 2](<http://artemis.ms.mff.cuni.cz/pogamut/tiki-index.php>) is an agent platform for Unreal Tournament, free for non-commercial use. They have recently made an announcement where they mention that Drools can be used:  
<http://www.java-forums.org/java-announcements/3205-pogamut-2-java-ide-creating-own-bots-ut2004.html>

I’m really excited to see that posting as I’ve always wanted to expand Drools as a platform so that it becomes useful for games programming – good to see that this is finally happening :)
