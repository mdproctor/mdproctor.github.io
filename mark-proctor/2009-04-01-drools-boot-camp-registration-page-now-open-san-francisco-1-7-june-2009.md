### [Drools Boot Camp Registration Page Now Open : San Francisco 1-7 June 2009](<https://blog.kie.org/2009/04/drools-boot-camp-registration-page-now-open-san-francisco-1-7-june-2009.html>)

by [Mark Proctor](<https://blog.kie.org/category/all?search_authors=3> "Posts by Mark Proctor") \- April 1, 2009  
[Rules](<https://blog.kie.org/category/rules>) [Article](<https://blog.kie.org/content_type/article>)

The Drools Boot Camp [registration page](<http://www.jboss.org/community/docs/DOC-13468>) is now open. If you are hoping to attend, please put your details there and send me an email, so that we can estimate and manage attendance. We only have spaces for about 20 or so people, so please do hurry. Registration wiki page can be found [here](<http://www.jboss.org/community/docs/DOC-13468>), you will need to login to edit the page.

The Boot Camp is schedule for the 1-7 of June 2009 at the Hilton San Francisco, and includes both time tabled sessions and well as all day hacking with all the Drools team. The event is free and open to the public. Full details of the event can be found [here](<http://blog.athico.com/2009/03/drools-boot-camp-san-francisco-june.html>).

A big thank you to [Third Pillar](<http://thirdpillar.com/>) for sponsoring the meeting room.
