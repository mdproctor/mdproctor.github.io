Some great external material is starting to appear around the web now for Drools 4.0

[Does your bad Java code need help? (from Rules and Flow) (Paul Browne)](<http://www.oreillynet.com/onjava/blog/2007/10/does_your_bad_java_code_need_h.html>)  
Gives an intro into the new rule flow stuff using the Number Guess example.

[Deploying JBoss Drools BRMS on Weblogic (Paul Browne)](<http://www.firstpartners.net/kb/index.php/Deploying_JBoss_Drools_BRMS_on_Weblogic>)  
Details how to get Drools setup and running on Weblogic

[Banking on DROOLS – Part 1 (John Dunning)](<http://droolsrules.blogspot.com/2007/10/banking-on-drools-part-1.html>)  
Detailed step by step tutorial on using Drools to solve simple bank account management problems.
