### [User and Group Management UI for Drools and jBPM workbench](<https://blog.kie.org/2015/08/user-and-group-management-ui-for-drools-and-jbpm-workbench.html>)

by [Mark Proctor](<https://blog.kie.org/category/all?search_authors=3> "Posts by Mark Proctor") \- August 14, 2015  
[Tools](<https://blog.kie.org/category/tools>) [Article](<https://blog.kie.org/content_type/article>)

Roger has been working on an SPI with a nice UI for the management of users and groups. Now people no longer have to go out to command line or some external tool to add and manage users and groups. We have a pluggable SPI approach with the main implementation currently targeting key cloak. We also have implementations for Wildfly and Tomcat properties files. This work should be part of our 6.4 release, later this year.

You can watch a youtube video [here](<https://www.youtube.com/watch?v=ioEG1ttMG4s>), don’t forget to turn on HD (there is no audio).
